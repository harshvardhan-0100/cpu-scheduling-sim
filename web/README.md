# CPU Scheduling Simulator — Web Frontend

> **For full project documentation, see the [main README](../README.md)**

This is the React frontend for the CPU Scheduling Simulator. 

## Quick Start

```bash
npm install
npm start
```

Opens on [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
```

The app is deployed on **[Vercel](https://cpu-scheduler-mu.vercel.app/)**
